 $ git config
---------------------------------
 Shows git config information

 $ git config --list
---------------------------------
 Shows git config lists
 
---------------------------------

 Configure the global username
 
---------------------------------
 $ git config --global user.email "hasan@gmail.com"
---------------------------------
 Configure the global email address
